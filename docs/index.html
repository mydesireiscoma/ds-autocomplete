DS Autocomplete
Dead simple autocomplete

#About

DS Autocomplete is a tiny, dependency-free, highly configurable blah blah blah. No need to describe how much this plugin is good. There is already tons of autocomplete plugins. Just check examples below and use it in your project, if it fits to your needs.

#Installation

Download plugin and include it in to your template
<script src="ds-autocomplete.dist.js"></script>
Or install the ds-autocomplete from the npm
npm install ds-autocomplete
And include it in your project
import DSAutocomplete from 'ds-autocomplete'

#Usage

Create a HTML input 
<input type="text" class="autocomplete" placeholder="Type something..."/>
Define a source for search.
It can be an Array: 
const items = ['one', 'two', 'three']
Or Object: 
const items = {
  1: { name: 'United Arab Emirates' }
  2: { name: 'China' },
  3: { name: 'United States of America' },
  4: { name: 'Canada' },
  5: { name: 'France' },
  // and so on
}
Or Function, what returns Object, Array or Promise: 
const items = async () => {
  const result = await axios.get('https://example.com')
  return result.data
}
Or Promise: 
const items = new Promise((resolve) => {
  setTimeout(() => {
    resolve(['one', 'two', 'three'])
  })
})
And initiate the plugin: 
const autocompleteInput = document.querySelector('.input')
const autocomplete = new DSAutocomplete(autocompleteInput, { items })

#Configuration

DSAutocomplete constructor takes up to three arguments:
  • Element
  • Main config
  • Keyboard config config
Where element - HTML Input element
Main config - object:
{
  // Autocomplete conif defaults
  // Search source
  items: [],
  // If true, results (if was previously fetched) will be shown when input takes focus
  showResultsOnFocus: false,
  // Object property should be used as value
  itemValuePropertyName: 'name',
  // Minimum characters required in input for trigger search
  minCharactersForSearch: 3,
  // Override, if you need custom search results item template
  itemTemplate: this.buildSearchItemTemplate,
  // Override, if you need custom search results container template
  resultsTemplate: this.buildSearchResultsTemplate,
  // Class, applied to search results item root element
  itemClass: 'autocomplete__result',
  // Class, applied to search results root element
  resultsClass: 'autocomplete__results',
  // Class, applied to result, when focused
  itemFocusedClass: 'autocomplete__result_focused',
  // Class, applied to input, when focused
  inputFocusedClass: 'autocomplete__input_focused',
  // Class, applied to search results root element, when hidden
  resultsHiddenClass: 'autocomplete__results_hidden'
}
Keyboard config - object:
{
  // Keyboard config defaults
  // Keys used for hide search results without selecting currently highlighted item
  cancel: [27],
  // Keys used for selection currently highlighted item
  select: [9, 13],
  // Keys, which are should not fire search results refetching
  ignored: [37, 39, 38, 40, 13, 27, 16, 9, 17, 18],
  // Keys used to navigate to the up on the search results
  up: [38],
  // Keys used to navigate to the down on the search results
  down: [40],
  // Keys should be prevented on search results selection
  prevent: [9, 13]
}

#Examples

Plain array

Example code:
<label for="autocomplete-plain-array">Search something:</label>
<input id="autocomplete-plain-array" type="text">
<script>
  const itemsPA = ['one', 'two', 'three']
  const autocompletePAInput = document.querySelector('#autocomplete-plain-array')
  new DSAutocomplete(autocompletePAInput, { items: itemsPA })
</script>

Promise

Example code:
<label for="autocomplete-promise">Search something:</label>
<input id="autocomplete-promise" type="text">
<script>
  const itemsP = (query) => {
    return new Promise((resolve, reject) => {
      const xhr = new XMLHttpRequest()

      xhr.open('GET', 'https://api.github.com/search/repositories?q=' + query)

      xhr.onload = function () {
        if (xhr.status === 200) {
          resolve(JSON.parse(xhr.responseText).items)
        } else {
          reject(new Error('Request failed'))
        }
      }

      xhr.send()
    })
  }
  const autocompletePInput = document.querySelector('#autocomplete-promise')
  new DSAutocomplete(autocompletePInput, { items: itemsP })
</script>
